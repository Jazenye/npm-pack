# 天气预报

天气预报的命令行工具

npm工具名： weather-jazen

## 安装：
```
npm install -g weather-jazen
```

## 使用方法
在命令行中键入：
```
weather
```

返回信息： 日期， 所在城市， pm2.5， 风向, 温度范围。

预报范围： 4天以内

### 查询目标城市天气
子啊命令行中键入
```
weather cityName
```
- cityName: 城市名称， 使用中文键入
